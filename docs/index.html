Stage

Stage 2 - MoreFix: plateforme e-commerce et gestion intelligente du stock

Synthese du stage 2: une solution e-commerce + admin stock exploitable en entreprise.

Contexte et besoin métier

MoreFix avait un stock gere manuellement et peu de visibilite web.

  • Inventaire non centralise
  • Aucune alerte automatique
  • Pas de vitrine e-commerce

Solution livrée

Plateforme en deux volets: vitrine e-commerce + administration stock.

  • Gestion CRUD produits, catégories et fournisseurs
  • Tableau de bord d'administration sécurisé
  • Recherche et suivi des stocks

Architecture technique

Architecture moderne orientee performance et maintenabilite.

Next.js React 19 TypeScript Tailwind CSS Firebase Auth Realtime Database Netlify React Hook Form + Zod

Module IA / NLP

Chatbot IA pour interroger le stock en langage naturel.

  • Stock faible, quantites, valeur d'inventaire
  • Extraction d'intentions et pattern matching
  • Reponses basees sur les donnees en temps reel

Qualité, sécurité et accessibilité

  • Validation multi-niveaux
  • Routes admin protégées et gestion de session
  • Tests CRUD et chatbot
  • Conformité WCAG 2.1 AA et interface responsive
